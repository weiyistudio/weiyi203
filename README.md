# 酒店预订系统的设计与实现
#### 介绍



#### 项目说明
本系统主要分为用户预订浏览模块和管理员管理信息模块。用户操作主要有：酒店信息浏览，用户注册，用户登录，预订房间，查看订单，查询会员中心等功能模块。管理员管理信息主要有：酒店信息管理，用户管理，订单管理等功能模块。
